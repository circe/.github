Provides [default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) across the [Circe GitHub org](https://github.com/typelevel).  The [profile directory](./profile) appears on the [GitHub org](https://github.com/circe).

This repository name and the filenames are magical.  Rename neither.

This repository is forked from the equivalent repository for Typelevel. Mergify is used to stay up-to-date with changes to the community health files in Typelevel.
